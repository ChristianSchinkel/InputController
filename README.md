# InputController

## _This library provides a typesafe approach for user input_

InputController is a tool for collecting input from the user in a CLI application. The developer using this controller can enforce the expected data type for input used by the program.

- Typesafe programming
- Ensures datatypes of `str`, `int`, `float`, and `bool`
- Ensures that the user enters only the data type the program requires

## Features

- Prompts the user for input and validates the entered datatype.

This project is implemented in Python.
